Authmol

Repositório

Desafio de um processo seletivo que eu participei.

O desafio consistia, entre outras coisas que não envolviam código, em criar uma página de cadastro que utilizasse a API da ViaCep.


Funcionalidades

  • Cadastro de um usuário com email, senha e endereço.
  • Confirmação de email.
  • Integração AJAX com a API da ViaCep para validação do endereço.
  • Feito com Domain Driven Design
