FitBoss é um programa demonstrativo de uma API de um sistema de gerenciamento de alunos e funcionários de uma rede de academias.

Implementa a Clean Architecture com CQRS. Feito com C#, .NET Core e SQLite e possui testes unitários e de integração.

Funcionalidades

  • Autenticação e autorização através de roles
  • Gerenciamento de alunos e funcionários

Instruções

Caso queira testar a API, siga as instruções abaixo para fazer login com o usuário de testes.


1. Clique no endpoint de login

2. Clique em Try it out

3. Clique em execute, o usuário e senha já vêm preenchidos

4. O resultado será um token e o id do usuário. Copie o token como mostra a imagem

5. Clique no botão Authorize

6. Na tela que aparecer, digite "Bearer " (com o espaço) e cole o token que você copiou. Depois clique em no botão Authorize.

7. Pronto! Agora você estará autorizado a utilizar as outras rotas da API.
