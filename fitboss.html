FitBoss

Repositório

FitBoss é um programa demonstrativo de uma API de um sistema de gerenciamento de alunos e funcionários de uma rede de academias.

Implementa a Clean Architecture com CQRS. Feito com C#, .NET Core e SQLite e possui testes unitários e de integração.

Funcionalidades

  • Autenticação e autorização através de roles
  • Gerenciamento de alunos e funcionários
